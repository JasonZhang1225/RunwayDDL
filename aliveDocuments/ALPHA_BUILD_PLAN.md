## 四、图片输入功能说明

**重要**：图片输入全链路属于 Alpha 必做功能

### 4.1 功能范围

| 子功能 | 说明 |
|--------|------|
| 相册选择 | 从系统相册选择图片 |
| 相机拍照 | 调用相机拍摄图片 |
| 权限处理 | 相册/相机权限请求和引导 |
| 图片预览 | 选择后显示缩略图 |
| 图片压缩 | 压缩后上传 AI |
| OCR 解析 | 调用多模态 AI 提取文字 |
| 图片存储 | 保存到应用目录 |
| 图片显示 | 在事项详情中显示图片 |

### 4.2 技术实现

- 使用 `image_picker` 插件
- 调用通义千问 VL 或文心一言多模态 API
- 图片保存到应用沙盒目录

---

## 五、XY 轴互换功能说明

**重要**：XY 轴互换属于 Alpha 后段功能

### 5.1 功能说明

- 默认视图：日期纵向 + 分类横向
- 互换视图：分类纵向 + 日期横向
- 切换按钮位于左上角

### 5.2 实现时机

- 必须在默认主视图稳定之后
- 不阻塞 Sprint 0-2 的基础开发
- 建议在 Sprint 3 后期或之后实现

---

## 六、筛选功能说明

**重要**：筛选功能不进入 Alpha，推迟到 Beta

### 6.1 Alpha 替代方案

| 筛选需求 | Alpha 替代方案 |
|----------|----------------|
| 时间范围筛选 | 主日期流固定展示"今天起未来 30 天" |
| 分类筛选 | 显示全部分类，无筛选 |
| 状态筛选 | 显示全部状态，未完成在前 |
| 逾期查看 | 逾期区固定展示在顶部 |
| 历史查看 | 历史区可展开查看 |

### 6.2 Beta 规划

筛选功能将在 Beta 版本实现，包括：
- 时间范围筛选（今天/近7天/近30天/全部）
- 分类筛选（点击分类表头）
- 状态筛选（全部/未完成/已完成）
